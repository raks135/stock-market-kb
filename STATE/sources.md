# Source Registry

Credibility tiers:
- Tier 1 — Authoritative: regulator/primary/academic (SEC, FINRA, NYSE papers, peer-reviewed microstructure, CFA curriculum).
- Tier 2 — Reputable practitioner: Investopedia, Optiver, Schwab/Vanguard/FINRA education, established blogs.
- Tier 3 — Community: forums/Medium — leads only, never sole source.
